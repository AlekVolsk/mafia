# Mafia

Free online tool for hosting Mafia (Werewolf) party games over video calls.

## Website

<https://mafia.alekvolsk.info/>

## About the Project

This resource is designed for the **game host**.
It allows roles to be distributed to players remotely without using physical cards. The game rules are included directly on the website.

The project is completely free and open to everyone.

## Translations

The interface is available in 12 international languages:
**English, Russian, Spanish, German, French, Italian, Turkish, Portuguese, Arabic, Hindi, Persian, Hebrew**

Translations were generated automatically.
If you notice an inaccuracy or mistake, please submit a [Pull Request](https://github.com/AlekVolsk/mafia/pulls).

## Privacy

The service does not collect, store, or process any personal user data.
It fully complies with European (GDPR) and U.S. data protection regulations.
