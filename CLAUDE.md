# Правила проекта

## Язык общения

Всегда отвечай на **русском языке**, независимо от языка вопроса.

## Стек

- PHP (index.php — точка входа, без фреймворков)
- Vanilla JS (assets/script.js, IIFE, без сборщиков)
- CSS (assets/style.css, CSS-переменные)
- SVG-спрайт (assets/sprites.svg, inline-инъекция через fetch)
- i18n через JSON-файлы (lang/*.json), 8 языков: en, ru, es, de, fr, it, tr, pt

## Соглашения

- SVG-иконки — только через `<use href="#icon-..."/>`, определяются в sprites.svg как `<symbol>`
- Цвета иконок — через CSS `color` на элементе `svg` или предке, все пути в спрайте используют `currentColor`
- Переводы — добавлять во все 8 локалей одновременно
- Без внешних зависимостей, без npm, без сборки
